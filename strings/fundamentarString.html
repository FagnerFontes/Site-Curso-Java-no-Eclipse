Strings em Java: Fundamentos


  1. Introdução às Strings

    2. Em Java, um string é uma sequência de caracteres usada para representar texto. A classe 'string' é imutável, o que significa que, uma vez criada, uma string não pode ser alterada. Qualquer operação que aprentemente modifica uma string, na verdade, cria uma nova string.
