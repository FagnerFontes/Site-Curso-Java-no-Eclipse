Literais


    Uma lieral de string em Java é uma sequência de caracteres delimitada por aspas duplas ('"'). Elas são usadas para representar valores constantes de 'strings' diferetamente no código.

    Caracteristivas das Literais de String:

  1. Imutabilidade:

    Literais de string são imutáveis. Isso significa que umavez criadas, seu valor não pode ser alterado.

  2. Pool de Strings:

    Java utiliza um pool de strings para otimizar o uso de memória. Qual uma literal de string é criada, o compilador verifica se uma instância com o mesmo valor já existe no pool. Se existir, a referência para essa instância é retornada. Se não, uma nova instância é criada e adicionada ao pool.

Literais de String e Novas Instâncias:
  1. Usando o Operador 'new':
    • 'str1' usa uma literal de string e aponta para a instância no pool.
    • 'str2' usa o operador 'new', criando uma nova instância no heap, fora do pool de strings.

Resumo

  • Literal de String:
    É uma sequência de caracteres entre aspas duplas ('"'). Ex.: 'Hello, Word"'.
  • Imutabilidade:
    Uma vez criada, a 'string' não pode ser alterada.
  • Pooling de Strings:
    Literais de string são armazenada em um pool para otimizar o uso de memória e melhorar o desempenho.
  • Operações:
    Literais de string podem ser concatenadas, comparadas e manipuladas com métodos da classe 'String'.
