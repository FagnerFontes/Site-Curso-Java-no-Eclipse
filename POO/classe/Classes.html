Programação Orientada a Objetos

Classes e Objetos


  • Definição de classe
    • É um modelo ou blueprint que define as propriedades atributos e comportamentos métodos comuns a todos os objetos do mesmo tipo. Uma classe pode ser vista como uma definição abstrata de um objeto.
      Uma classe em Java é um "blueprint" ou modelo a partir do qual objetos são criados. Ela define um conjunto de atributos (variáveis de instância) e métodos (funções) que são comuns a todos os objetos do mesmo tipo.
  • Definição de classe
    • Nome da classe: Idendificador unica da classe.
    • Atributos (campos) : Variáveis que representam as propriedades ou característivas da classe.
    • Métodos: Funções que definem os comportamentos da classe.
    • Construtores: Métodos especiais que são chamados quando um objeto é instanciado, utilizados inicializar os atributos da classe.
    • Modificadores de Acesso: Controlam a visibilidade e o acesso aos componentes da classe (Ex.: 'public', 'private', protected)
Abaixo a estrutura de uma classe em Java
        // Declaração da classe
public class NomeDaClasse {
    // Atributos (Campos)
    private Tipo atributo1;
    private Tipo atributo2;

    // Construtor
    public NomeDaClasse(Tipo parametro1, Tipo parametro2) {
        this.atributo1 = parametro1;
        this.atributo2 = parametro2;
    }

    // Métodos
    public Tipo getAtributo1() {
        return atributo1;
    }

    public void setAtributo1(Tipo atributo1) {
        this.atributo1 = atributo1;
    }

    public void metodoExemplo() {
        // Implementação do método
    }
}
